Code Faster: Convert Keystrokes into MOS Objects
KGO Basic AutoHotkey Builder
Created By: Kurt.Stoneburner@abc.com

This software builds Autohotkey scripts that turn keystrokes into MOS commands. Keystrokes are used instead of dragging and dropping in scripts.

This works by dragging MOS objects from the Overdrive Server webpage: http://OverDrive_Server_IP/newsroomplugin/ and dragging it onto a hotkey line, which assigns a keyboard Key plus any combination of SHIFT,CTRL,WIN and ALT.

This is a very basic builder designed to give maximum benefit for the least effort. Autohotkey can be used to do a whole lot more.

1.) Download and install the latest version of www.autohotkey.com

2.) Drag desired MOS Element onto a line.

3.) Assign a Key or special key command for F Keys, numpad, etc..(see Below)

4.) Click the appropriate Checkbox if SHIFT,CTRL,WIN or ALT is to be pressed with the Key

5.) Click New Hotkey to add a new blank hotkey line

6.) Repeat


Build the AHK file

1.) Click Export AHK and save the file

2.) Open the Download Folder and Double-Click Macros.AHK to launch the AutoHotkey Script.

Note: Scripts built with Export have a reload function built in. Press SHIFT+CTRL+WIN+ALT+R will reload the autohotkey script. This it is useful when making later changes to the script. Notably, the script can be editted in notepad, saved, then reloaded with SHIFT+CTRL+WIN+ALT+R


Copy AHK to Clipboard

This button builds a basic Autohotkey script and places it in the clipboard. This is designed to add new autohotkeys to an existing script.


Saving your progess

You can also save your work by pressing CTRL+S and saving the HTML file. This can be opened later in a web browser


Special Keys

Keys other than letters and numbers may be used.

F keys are F1,F2,F3-F12.

The Number Pad is: Numpad1,Numpad2,Numpad3-Numpad9

A Complete keylist can be found at https://www.autohotkey.com/docs/KeyList.htm



To Save your progress use Save Hotkeys.
To load saved work, use Load Hotkeys
